### 前言
`React-Redux`是`Redux`的官方`React`绑定库。它能够使你的`React`组件从`Redux store`中读取数据，并且向`store`分发`actions`以更新数据

### 安装
```powershell
$ npm install --save react-redux
$ yarn add react-redux
```
