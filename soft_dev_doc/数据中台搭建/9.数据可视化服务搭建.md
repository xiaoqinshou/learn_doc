### 数据可视化服务搭建
* 选用 datart 进行可视化分析
* [datart 库](https://github.com/running-elephant/datart/)
* [官方文档](https://running-elephant.github.io/datart-docs/docs/)

#### datart 部署
##### 创建空间
```sh
$ kubectl create namespace datart-space
```
##### 创建config配置文件
```yaml
kind: ConfigMap
apiVersion: v1
metadata:
  name: datart-configs
  namespace: datart-space
  labels:
    app.kubernetes.io/name: datart
    app.kubernetes.io/version: 1.0.0-rc.2
data:
  common_properties: >-
    #

    # 应用数据库配置

    # 数据库IP或域名

    datasource.ip=172.17.0.8      

    # 数据库端口

    datasource.port=3306          

    # 数据库名称

    datasource.database=datart    

    # 用户名

    datasource.username=root      

    # 密码

    datasource.password=Fx@Sjzt2023     

    # 应用服务器配置

    # 服务器端口

    server.port=80              

    # 服务器地址（内网地址）

    server.address=0.0.0.0        

    # datart 全局配置

    # 应用主页地址（公网地址）

    datart.address=http://127.0.0.1:8080                

    # 注册账户时是否需要邮件激活

    datart.send-mail=false                             

    # ChromeDriver 地址（用于截图）

    datart.webdriver-path=http://127.0.0.1:4444/wd/hub  
```

##### 部署
* pod
```yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: datart
  namespace: datart-space
  labels:
    app.kubernetes.io/name: datart
    app.kubernetes.io/version: 1.0.0-rc.2
spec:
  replicas: 1
  selector:
    matchLabels:
      app.kubernetes.io/name: datart
  template:
    metadata:
      labels:
        app.kubernetes.io/name: datart
    spec:
      containers:
      - name: datart
        image: datart/datart
        ports:
          - containerPort: 80
            name: web-port
            protocol: TCP
        volumeMounts:
        - name: mysql-conf
          mountPath: /datart/config/datart.conf #容器内的挂载点
          subPath: common_properties
      volumes:
      - name: mysql-conf            #指定名称必须与上面一致
        configMap:
          name: datart-configs      # 和configmap 名称一样即可
      restartPolicy: Always
```
* service
```yaml
apiVersion: v1
kind: Service
metadata:
  name: datart-service
  namespace: datart-space
spec:
  selector:
    app.kubernetes.io/name: datart
  type: NodePort
  ports:
  - name: web-port
    protocol: TCP
    port: 80
    nodePort: 30002
    targetPort: web-port
```

#### 访问
* 之前开放了固定端口, 直接浏览器访问固定端口30002即可
